
# To-Do List API

## Project Description
This is a simple REST API built with Node.js and Express for the New Generation Academy homework. It allows users to manage a to-do list with the following features:
- View all tasks (GET)
- Add a new task (POST)
- Update a task (PUT)
- Modify a task (PATCH)
- Remove a task (DELETE)

## How to Run the Server
1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install

```

3. Start the server:
```bash
node index.js

```


4. The server runs on `http://localhost:3000`.
